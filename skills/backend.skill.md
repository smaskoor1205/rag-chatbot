# Backend Skill

- Express architecture: layered routes, controllers, services, repositories, middleware, and typed configuration.
- API standards: JSON APIs with predictable response envelopes, validation, pagination, and explicit status codes.
- Validation: Zod schemas at system boundaries for request bodies, params, query strings, and environment variables.
- Error handling: centralized error middleware with safe public messages and structured logs.
- Authentication: JWT access tokens, bcrypt password hashing, role checks, and authenticated request context.
- Security: Helmet, CORS allowlists, upload limits, input validation, and least-privilege filesystem access.
