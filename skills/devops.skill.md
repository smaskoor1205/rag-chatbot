# DevOps Skill

- Docker standards: multi-stage builds, non-root runtime users, health checks, and persistent volumes for data.
- GitHub Actions: install, lint, test, build, and Docker validation on pull requests and main.
- CI/CD: reproducible npm installs, separate verification jobs, and environment-driven deployments.
- Logging: structured logs to console and file with request ids and safe metadata.
- Monitoring: health and readiness endpoints, request logging, and application-level operational counters.
- Deployment: documented compose workflow with persistent uploads, SQLite, vector store, and logs volumes.
- Security hardening: secrets through environment variables, dependency audit, upload limits, and minimal containers.
